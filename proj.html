  • Projects Done

    Tetris

    Spring 2016

    Prof. Amitabha Sanyal


    Made a game of Tetris using Racket (a functional programming language). Users can control the speed and the position of the falling blocks generated. The game also features a real time loading screen.


    Line Follower Bot

    Spring 2016

    E and RC Club, IIT Bombay


    Made a Line Follower bot by programming an Arduino and using IR sensors to check for the white line. DC motors were used to drive the bot.


    RC Plane

    Autumn 2016

    Aeromodelling Club, IIT Bombay


    Made a remote-controlled airplane using an ESC (Electronic Speed Controller) that handles the amperage draw of the motor. The plane body was made using a plastic called coroplast. A BLDC motor was used for the main rotor and servomotors were used to control the airfoil angle.


    RC Racing Bot

    Autumn 2016

    E and RC Club, IIT Bombay


    Made a remote-controlled bot which received signals using HC-05 via bluetooth from our mobile phone and processed it using an ATTiny chip (pre-programmed) to control our bot.
